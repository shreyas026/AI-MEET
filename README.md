# AI-MEET

AI Meeting Operator is a TanStack Start application backed by Supabase and Gemini API helpers.

## Project Structure

```text
AI-MEET/
  frontend/   TanStack Start React app
  backend/    Supabase schema/config and backend docs
```

## Start The Project

```bash
cd frontend
npm install
npm run dev
```

Create `frontend/.env` with the required Supabase and Gemini environment variables before running locally.
